模块

每个 .py 文件的主文件就是模块名称，想要导入模块时必须使用 import 关键词来制定模块名称。若要引用模块中定义的名称，则必须在名称前加上模块名称与一个 “.“ 符号。

pycache

CPython 将 .py 文件编译后的字节码文件，如果之后再次导入同一个模块，检测到源码文件没有更改过，就不会再对源码重头开始进行语义和语法解析等操作。
