Apple Store 分享（未定稿）

May 23, 2019 | Hits

大家好～今天我带来的分享主题是《Apple 带给我的成长》。用这短短的十几分钟来回顾一遍 Apple 对我的影响。

我来自海南，网络 ID 是 pjhubs。非常喜欢在 Apple 生态圈中实现自己的创意，我还是一名开源爱好者，经常会做一些好玩的 app。最近我给自己的大学画上了一个圆满的句号，我拿到了梦寐以求的 WWDC 奖学金，这个是我的博客。

Apple 给我的影响主要集中在以下五个时间节点中，每一个节点都帮助了我，让我得到了很大的改变。

当我刚上大一的时候，我对身边的一切都感到害怕，不知从何下手，后来从社团了解到了还有 iOS 开发这样的事情后我非常激动，开始正式投入了学习 iOS 开发中，也可以说，我把对自己不自信状态进行了转移，让我自己慢慢改正了对自己的看法。

在练习的过程中，我慢慢产生了对模拟器的不满，开始找身边的同学帮忙，找他们每人借了 100 块凑够了购买了 iPod touch 6 的钱，它成为了我第一款 iOS 设备。

放假回家的时候，我吭哧吭哧的凭借当时自己所学到的 iOS 开发知识，完成了一个自动出题器，具备自动刷新四则运算的出题器，做完了以后立马拿给了家里的弟弟妹妹们玩，看着它们争先恐后的样子我很开心。

经过了一年多的学习后，我开始做更多完整的产品，包括游戏。当时为了参加首届中国iOS应用开发大赛我总共熬了一个多星期的夜耗时整整一个月的时间，把这个针对大学生学习生活的应用做出来了，最后很开心的通过这个作品拿到了属于我的第一台 iPad。

这是我在 iOS 上做的第一款游戏，它发挥了更多 iPhone 的硬件能力，同时也让我对游戏开发有了更多的了解，明白了如何从 0 到 1 做出一款游戏。

慢慢的我到了要找实习的时间，通过之前学习 iOS 开发的经历，我很容易就找到了实习。我的大部分实习时间是在滴滴度过的，在滴滴我明白了做好一件事情的正确流程应该是怎么样的，了解到了如何与团队中的其它同学进行有效的沟通，对细节的追求更加彻底。我开始变得稳重，开始变得更加自信，同时也明白了自己写下的每一行代码都会影响到实际的用户。

Github 是我的第二个大学。github 是一个众多编程爱好者们分享代码的平台，在 github 上我们可以非常容易的看到其它人写的代码。我通过 github 学习到了非常多对我有帮助的知识，这加速了我的成长。从社区中得到了成长，同时也在回报社区，我把自己之前做过的一些好玩的 app 进行了开源，通过代码的方式与大家分享我的知识。

针对今年的 WWDC 奖学金，我做了一个与黎族织锦有关的拼图小游戏。提问：大家有知道的关于黎族更多的文化习俗么？比如他们有哪些节假日？生活习惯是什么？

我抓住了黎族的织锦这个主题，黎族人非常崇拜自然，他们崇拜风雨雷电，崇拜猪马牛羊，甚至还崇拜力量本身。这是我当时做产品设计进行头脑风暴的草稿。

现在，我要给大家试一下这个小游戏～（解释）

在这个小游戏当中，为了增加更多的趣味性，我给每一块拼图都加上了“吸附”效果，类似于磁铁。如果能够成功拼接出大力神，大力神将会“动”起来，并放出彩纸。

我大一的时候因为害怕和担心，我总是担心自己没法融入新集体，没法去找到自己喜欢的东西，感谢 Apple 让我找到了自己真正喜欢的东西，让我锻炼出了自我探索的能力，现如今我正迈向自我实现的道路上，感谢 Apple。

这是我和其它几位小伙伴一块做的 app，名字叫 vary，我们结合了大量的动画和设备的硬件能力，目前还在努力的打磨中，相信大家一定会喜欢的！
